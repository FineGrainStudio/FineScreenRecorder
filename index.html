Infatica P2B Network Participation

By using this app you will become a peer on the Infatica P2B network. This means that a tiny bit of your idle Internet bandwidth resources can be shared with this network. The P2B network spreads its traffic through millions of idle peers and therefore has minimal effect on total bandwidth consumption.

Please note that NONE of your personal information is accessed and NO USER DATA is collected or shared with external parties except for the IP location data.
